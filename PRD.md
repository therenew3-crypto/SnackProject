## 1. 제품 개요 (Overview)

- **제품명**: 타임머신 IF 계좌 계산기 (가칭)
- **한 줄 요약**: 유저가 특정 과거 시점에 특정 자산(주식·코인·금)을 매수해 원하는 시점에 매도했다면 얻었을 **가상 수익·자산 잔액**을 5초 만에 계산해 주는 스낵형 웹 서비스.
- **핵심 목적**: 대중의 '껄무새'(그때 살걸…) 심리를 자극해 폭발적인 바이럴(카카오톡 공유)을 유도하고, 결과 직전 전면 광고로 수익을 창출.

## 2. 타깃 유저 및 핵심 가치 (Target & Value)

- **타깃**: 주식·비트코인 투자 경험이 있거나 관심 있는 2040 대중.
- **유저 가치**: "내가 그때 샀더라면 지금 얼마였을까?"라는 원초적 호기심을, **실제 시세 데이터** 기반 결과 카드 + **수익률 순위/등급/재치 멘트**로 5초 만에 시각적으로 충족.
- **차별점**: 단순 계산을 넘어 ① 전체 시나리오 중 내 **수익률 순위(상위 %)**, ② 수익률 구간별 **랜덤 재치 멘트·등급**으로 재미와 공유 욕구를 극대화.

## 3. 핵심 기능 요구사항 (Core Features)

### [F1] 가상 포트폴리오 입력 화면

- **자산 선택 (Dropdown, 카테고리 그룹)** — 표시 순서: 한국주식 → 미국주식 → 코인 → 금·원자재
  - 🇰🇷 한국주식: 삼성전자(005930), SK하이닉스(000660)
  - 🇺🇸 미국주식: 엔비디아(NVDA), 테슬라(TSLA), 애플(AAPL)
  - 🪙 코인: 비트코인(BTC), 이더리움(ETH), 도지코인(DOGE)
  - 🥇 금·원자재: 금(Gold)
  - **기본 선택값: SK하이닉스(000660)**
- **과거 매수 시점 (Dropdown)** — 구체적 날짜 + 스토리 라벨 10종
  - 2015.01.02 (비트코인 창세기) / 2017.12.17 (1차 코인 광풍 꼭지) / 2018.12.15 (크립토 겨울 바닥) / 2020.03.23 (코로나 대폭락) / 2021.11.09 (역대급 불장 꼭지) / 2022.01.03 (2022 상투) / 2022.11.09 (FTX 붕괴) / 2023.01.02 (약세장 신년) / 2024.01.02 (AI 랠리 직전) / 2025.01.02 (작년 신년)
- **매수 금액 입력 (Input)** — 원화 직접 입력 + 퀵버튼(100만 / 1,000만 / 1억 / 10억), 자동 천단위 콤마
- **매도 시점 (Slider)** — 2016년 말 ~ 2025년 말(연말) + 2026년 6월(현재), 총 11단계
- **입력값 초기화 버튼** — 처음 랜딩 상태로 리셋(자산=SK하이닉스, 시점=2015.01, 금액 비움, 매도=현재)

### [F2] 도파민 유발 결과 화면

- **최종 자산 잔액 산출**: `매수금액 × (매도가 / 매수가)` — 비율 계산이라 자산별 통화 단위 무관
- **카운트업 애니메이션**으로 금액 공개, 조/억/만원 단위 한글 포맷
- **수익률 배지**: 수익률 %, 배수, 상승(빨강)/하락(파랑)
- **수익률 순위**: 전체 유효 시나리오(자산×매수×매도 조합) 중 **n등 / 상위 n%**
- **수익률 구간별 등급(이모지 + 랜덤 명칭)** — 7구간
  - 💀 반토막↓ → 📉 손실 → 😐 본전(인생여전) → 🙂 쏠쏠 → 🔥 대박 → 🚀 인생역전 → 👑 전설
- **수익률 구간별 재치 멘트(랜덤, 약 70종)** — 같은 결과도 매번 다른 멘트
  - 예) 전설: "타임패러독스 유발자! 당장 사직서를 인쇄하고 회장님 뺨을 때리러 가세요."
  - 예) 본전: "인생여전! 당신의 인생은 변화가 없습니다."
- **컨페티 연출**: 3배 이상(대박 등급↑) 시 색종이 효과
- **공유 기능**: 결과를 담은 링크(`?a&b&amt&s`)를 공유 → 친구가 누르면 동일 결과 자동 표시

### [F3] 예외 처리 (Guard)

- 매수 시점에 자산이 미존재(상장·출시 이전, 데이터 이전)면 안내
- 매도 시점이 매수 시점보다 빠르거나 같으면(타임 패러독스) 차단
- 금액 미입력 시 안내

## 4. 비즈니스 모델 및 광고 연동 (Monetization)

- **광고 네트워크**: 카카오 애드핏 (국내 디스플레이 광고)
- **광고 영역 3종**
  | 위치 | 사이즈 | 노출 | 상태 |
  |---|---|---|---|
  | 상단 배너 | 320×100 | 항상 노출 | ✅ 정상 |
  | 하단 배너 | 300×250 | 항상 노출 | ✅ 정상 |
  | 전면(팝업) | 250×250 | 결과 직전 3초 인터스티셜 | ⏳ 충전/검증 중 |
- **결과 직전 전면광고**: 결과 확인 클릭 → "결과 분석 중…" 3초 카운트다운 모달 → 결과 공개
- **기술 메모**: 애드핏 SDK(`ba.min.js`)는 중복 로드 가드(`if(!("adfit" in self))`)가 있어 동적 추가 광고 렌더가 까다로움 → 광고를 페이지 로드 시 미리 배치하고 모달은 opacity로 토글하는 방식 채택. 미충전(노필) 시 영역 자동 접힘/폴백 처리.

## 5. 데이터 구조 및 로직 (Data & Logic)

- **데이터 방식**: 외부 API 실시간 호출 없이, 각 자산의 **시점별 실제 시세**를 코드 내부에 하드코딩(JSON/객체)하여 속도 극대화.
- **시세 출처 (2026-06-27 기준 실데이터)**
  - 미국주식·코인: `yfinance` 수정종가(액면분할·배당 반영)
  - 한국주식: `pykrx` (KRX 공식) 종가
  - 금: `yfinance GC=F` (USD/oz)
- **데이터 수집 스크립트** (재현·갱신용)
  - `fetch_prices.py` (코인·미국주식 매수 시점), `fetch_kr.py` (한국주식), `fetch_sells.py` (연말 매도가), `fetch_gold.py` (금), `og_image.py` (OG 썸네일 생성)
- **데이터 한계**: ETH·DOGE의 2015.01은 야후 데이터 시작(2017-11) 이전이라 null. 2025~2026 수치는 수집 시점 실데이터.

## 6. 공유 / SEO (Virality)

## 7. 기술 스택 및 UI/UX (Tech Stack)

- **프론트엔드**: 단일 HTML 파일 + Tailwind CSS(CDN) + Vanilla JS, Pretendard 폰트
- **디자인 컨셉**: 토스/증권사 앱풍 다크모드, 포인트 컬러 골드(#FFD700) + 상승 빨강/하락 파랑
- **배포**: Netlify (`if-ggeolmusae.netlify.app`), 소스 GitHub(`therenew3-crypto/SnackProject`)
- **배포 방식**: `netlify deploy`(드래프트) → `netlify api restoreSiteDeploy`(프로덕션 게시). 향후 GitHub 연동 CI 자동배포 검토.
